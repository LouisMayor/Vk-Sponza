# Vk-Sponza

Sponza scene using Vulkan

### Features:
- Camera
- Multi-texture
- ImGui
- Obj model loading via TinyObjLoader
- Texture loading via STB
- Supports multisampling
- Supports mipmaps
- Pipeline caching
